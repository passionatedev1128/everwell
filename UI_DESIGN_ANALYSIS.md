# UI/UX Design Analysis - Competitor Websites

## Website Analysis: Click Cannabis, Blis Brasil, Cannect Life

---

## 1. CLICK CANNABIS (clickcannabis.com)

### Color Palette
- **Primary Green**: `#285e31` (rgb(40, 94, 49)) - Trust, nature, medical
- **Background**: `#ffffff` (Pure white)
- **Text**: `#000000` (Black)
- **Overall**: Clean, medical, professional aesthetic

### Typography
- **Font Family**: `Inter, system-ui, arial`
- **Style**: Modern sans-serif, highly readable
- **Usage**: Professional, clean, medical authority

### Layout & Structure
- **Navigation**: Horizontal menu, clean header with logo
- **Sections**: Well-organized with clear hierarchy
- **Spacing**: Generous padding, breathing room
- **Max Width**: Full-width sections with centered content

### Key UI Elements
- ✅ Clean, minimal navigation
- ✅ Professional header with logo
- ✅ Clear section divisions
- ✅ Medical/trustworthy aesthetic
- ✅ Good use of white space

### Strengths
1. **Trustworthy Design**: Green conveys health and nature
2. **Clean Layout**: Easy to navigate, professional
3. **Medical Authority**: Professional typography and spacing
4. **Readability**: High contrast, clear hierarchy

## Typography System

### Primary Font
- **Font Family**: `Inter, -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif`
- **Why**: Professional, highly readable, medical authority (from Click Cannabis)
- **Alternative**: `Poppins` for more modern, friendly feel (from Blis)

### Font Hierarchy
- **H1 (Hero)**: 48px, Bold, `#2c3e50`
- **H2 (Section Headings)**: 36px, Semi-bold, `#2c3e50`
- **H3 (Subheadings)**: 28px, Semi-bold, `#2c3e50`
- **H4**: 24px, Medium, `#2c3e50`
- **Body**: 16px, Regular, `#5a6c7d`
- **Small**: 14px, Regular, `#5a6c7d`
- **Button**: 16px, Semi-bold, `#ffffff`

## Layout Patterns

### Header/Navigation
- **Style**: Clean, horizontal navigation (Click Cannabis approach)
- **Logo**: Left-aligned
- **Menu**: Center or right-aligned
- **Trust Badges**: Small badges in header (Blis approach) - "100% Legal", "Anvisa Approved"
- **User Account**: Right side, login/register buttons

### Hero Section
- **Style**: Large banner with bold headline (Blis approach)
- **Background**: White or soft beige (`#f5f3ef`)
- **CTAs**: Primary button prominently displayed
- **Image**: Hero image or illustration on right side

### Product Sections
- **Layout**: Grid layout (3 columns on desktop, 1 on mobile)
- **Cards**: White background, subtle shadow, rounded corners
- **Spacing**: Generous padding (Click Cannabis approach)
- **Hover Effects**: Subtle lift/shadow on hover

### Trust Indicators
- **Top Banner**: Benefit badges (Cannect approach)
  - "Frete grátis acima de R$X"
  - "Parcelamento em até Xx"
  - "100% Legal e Aprovado"
- **Section Badges**: Throughout site (Blis approach)

### CTAs (Call-to-Actions)
- **Primary CTA**: Green button (`#2d7d7d`) with white text
- **Secondary CTA**: Outlined button with green border
- **Placement**: 
  - Hero section: Large, prominent
  - Product cards: "Comprar" button
  - Throughout: "Agendar Consulta" button

## Component Design System

### Buttons

#### Primary Button
```css
background: #2d7d7d
color: #ffffff
padding: 14px 32px
border-radius: 8px
font-weight: 600
font-size: 16px
transition: all 0.3s ease
hover: background: #236666, transform: translateY(-2px), box-shadow: 0 4px 12px rgba(45, 125, 125, 0.3)
```

#### Secondary Button
```css
background: transparent
color: #2d7d7d
border: 2px solid #2d7d7d
padding: 12px 30px
border-radius: 8px
hover: background: #2d7d7d, color: #ffffff
```

## Responsive Design

### Breakpoints
- **Mobile**: < 768px
- **Tablet**: 768px - 1024px
- **Desktop**: > 1024px

### Mobile-First Approach
- Start with mobile design
- Progressive enhancement for larger screens
- Touch-friendly buttons (min 44x44px)
- Hamburger menu for mobile
